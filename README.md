# MultidimensionalArrayCrossGroup
多维数组交叉组合
如：[['a', 'b'], [1, 2, 3], ['hello', 'world']]
组合后变为：
['a', 1, 'hello']
['a', 1, 'world']
['a', 2, 'hello']
['a', 2, 'world']
['a', 3, 'hello']
['a', 3, 'world']
['b', 1, 'hello']
['b', 1, 'world']
['b', 2, 'hello']
['b', 2, 'world']
['b', 3, 'hello']
['b', 3, 'world']
